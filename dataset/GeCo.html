Data Set: GeCo - data Generator Corruptor

Proposed by: Peter Christen

Added on: 18 November 2016.

Tags: artificial, personal, external, public, generator

Description

The GeCo data generator and corruptor is an open source Python package that allows flexible generation or realistic personal information.

An online demo, Python program, and documentation are available.
